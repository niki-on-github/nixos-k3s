# [Mail-Archiver](https://github.com/s1t5/mail-archiver)

 Mail-Archiver is a web application for archiving, searching, and exporting emails from multiple accounts. Featuring folder sync, attachment support, mailbox migration and a dashboard. 

## Mail Accounts

### Gmail

- Provider: IMAP
- IMAP Server: `imap.gmail.com`
- IMAP Port: 993
- Username: Your E-Mail Address
- Password: Create and use the app password from `https://myaccount.google.com/apppasswords`.
